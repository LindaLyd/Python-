# 一、变量命名规则

## 1. 基本规则（必须遵守）

| 规则 | 正确示例 | 错误示例 |
|------|----------|----------|
| **只能包含**：字母、数字、下划线 | `user_name`, `age1` | `user-name`, `@name` |
| **不能数字开头** | `name1` ✅ | `1name` ❌ |
| **不能是关键字** | `class_` ✅ | `class` ❌ |
| **区分大小写** | `age` ≠ `Age` | - |

**Python 关键字（35个）：**
```
False, None, True, and, as, assert, async, await, break, class, 
continue, def, del, elif, else, except, finally, for, from, global,
if, import, in, is, lambda, nonlocal, not, or, pass, raise,
return, try, while, with, yield
```

## 2. 命名规范（PEP8推荐）

| 类型 | 规范 | 示例 |
|------|------|------|
| **变量名** | 小写+下划线 | `user_name`, `total_count` |
| **常量** | 全大写+下划线 | `MAX_SIZE`, `PI` |
| **类名** | 驼峰式（首字母大写） | `UserInfo`, `CarModel` |
| **函数名** | 小写+下划线 | `get_user()`, `calculate_total()` |
| **私有变量** | 前加单下划线 | `_internal_use` |
| **魔术变量** | 前后双下划线 | `__init__`, `__name__` |

## 3. 不能用的名字

```python
# ❌ 不要用内置函数名做变量
list = [1,2,3]        # 覆盖了 list() 函数
str = "hello"          # 覆盖了 str() 函数
print = "打印"         # 覆盖了 print() 函数

# ✅ 应该用
my_list = [1,2,3]
my_str = "hello"
print_func = "打印"
```

**常见被覆盖的内置函数：** `list`, `str`, `int`, `dict`, `set`, `min`, `max`, `sum`, `len`, `input`, `print`, `open`
