# 一、Python中的数值类型

Python主要有三种数值类型：

| 类型 | 名称 | 示例 | 说明 |
|------|------|------|------|
| **int** | 整数 | `42`, `-5`, `0`, `1_000_000` | 不限大小 |
| **float** | 浮点数 | `3.14`, `-0.5`, `1e3` | 双精度浮点 |
| **complex** | 复数 | `3+4j`, `5j`, `complex(1,2)` | 少考 |

---

# 二、整数（int）

## 1. 特点

- **不限大小**（可以任意大）
- 支持多种进制

```python
# 十进制（默认）
a = 42

# 二进制（0b开头）
b = 0b101010  # 42

# 八进制（0o开头）
c = 0o52      # 42

# 十六进制（0x开头）
d = 0x2A      # 42

# 数字分隔符（Python 3.6+）
e = 1_000_000    # 1000000，更易读
f = 0x_FF_FF     # 65535
```

## 2. 整数的特殊方法

```python
# 返回整数的字节长度
(42).bit_length()      # 6（二进制101010是6位）

# 判断是否为整数
isinstance(42, int)    # True
type(42) is int        # True
```

---

# 三、浮点数（float）

## 1. 特点

- **双精度浮点数**（64位）
- **精度问题**（二进制无法精确表示某些小数）

```python
# 小数形式
a = 3.14
b = -0.5
c = .5        # 0.5（小数点前可以省略0）

# 科学计数法
d = 1.2e3     # 1200.0
e = 5E-2      # 0.05
```

## 2. 特殊浮点数

```python
# 无穷大
float('inf')      # inf
float('-inf')     # -inf

# 非数字
float('nan')      # nan

# 判断特殊值
import math
math.isinf(x)     # 是否是无穷大
math.isnan(x)     # 是否是非数字
```

## 3. 浮点数精度问题（重点坑）

```python
# 经典问题
print(0.1 + 0.2)          # 0.30000000000000004（不是0.3！）
print(0.1 + 0.2 == 0.3)   # False（坑！）

# 解决方法1：四舍五入
round(0.1 + 0.2, 1)       # 0.3

# 解决方法2：使用 Decimal（精确计算）
from decimal import Decimal
Decimal('0.1') + Decimal('0.2')  # Decimal('0.3')

# 解决方法3：允许误差
abs((0.1 + 0.2) - 0.3) < 1e-9    # True
```

## 4. 浮点数转换

```python
# 整数转浮点
float(42)           # 42.0

# 字符串转浮点
float('3.14')       # 3.14
float('1.2e3')      # 1200.0

# 会报错的情况
float('abc')        # ValueError
float('1,234')      # ValueError（不能有逗号）
```

# 四、复数（complex）- 了解即可

```python
# 创建复数
c1 = 3 + 4j
c2 = complex(3, 4)   # 3+4j

# 获取实部和虚部
c1.real    # 3.0
c1.imag    # 4.0

# 共轭复数

```

# 五、数值运算

## 1. 基本运算符

| 运算符 | 含义 | 示例 | 结果 |
|--------|------|------|------|
| `+` | 加法 | `5 + 3` | `8` |
| `-` | 减法 | `5 - 3` | `2` |
| `*` | 乘法 | `5 * 3` | `15` |
| `/` | 除法（浮点） | `5 / 2` | `2.5` |
| `//` | 整除 | `5 // 2` | `2` |
| `%` | 取余 | `5 % 2` | `1` |
| `**` | 幂运算 | `5 ** 2` | `25` |

## 2. 类型转换（混合运算）

```python
# 混合运算时，Python会自动提升类型
# bool → int → float → complex

# int + float → float
3 + 2.5          # 5.5

# int + bool → int
5 + True         # 6（True=1）
5 + False        # 5（False=0）

# float + complex → complex
2.5 + (3+4j)     # (5.5+4j)
```

## 3. 常用数学函数

```python
import math

# 取整
math.floor(3.7)      # 3（向下取整）
math.ceil(3.2)       # 4（向上取整）
round(3.5)           # 4（四舍五入，银行家舍入法）
round(3.4)           # 3
int(3.7)             # 3（截断取整）

# 绝对值
abs(-5)              # 5
abs(3+4j)            # 5.0（复数的模）

# 幂运算
pow(2, 3)            # 8
math.sqrt(16)        # 4.0

# 三角函数
math.sin(math.pi/2)  # 1.0

# 常量
math.pi              # 3.141592653589793
math.e               # 2.718281828459045
```

# 六、类型判断和转换

## 1. 类型判断

```python
# type() - 精确判断
type(42) is int           # True
type(3.14) is float       # True

# isinstance() - 推荐（支持继承）
isinstance(42, int)       # True
isinstance(3.14, (int, float))  # True（判断是否整数或浮点）

# 判断是否为数字（整数+浮点+复数）
from numbers import Number
isinstance(x, Number)     # True if x是任何数字类型
```

## 2. 类型转换

```python
# 转整数
int(3.14)          # 3（截断，不四舍五入）
int("42")          # 42
int(True)          # 1
int("1010", 2)     # 10（二进制字符串转整数）

# 转浮点
float(3)           # 3.0
float("3.14")      # 3.14
float("1e3")       # 1000.0

# 转布尔（任何数字都可以转布尔）
bool(0)            # False
bool(0.0)          # False
bool(42)           # True
bool(-1)           # True
bool(float('nan')) # True（nan也是True！）
```

# 七、经典陷阱题

## 陷阱1：浮点数精度
```python
# 题目
0.1 + 0.2 == 0.3  # False！

# 原因：二进制无法精确表示0.1
print(0.1)        # 0.1（显示时四舍五入了）
print(format(0.1, '.20f'))  # 0.10000000000000000555
```

## 陷阱2：取整方向
```python
# 整除 // 是向下取整（不是向零取整）
print(5 // 2)     # 2
print(-5 // 2)    # -3（-2.5向下取整是-3，不是-2）
print(5 // -2)    # -3

# int() 是向零取整（截断）
print(int(5.7))   # 5
print(int(-5.7))  # -5（向零）
```

## 陷阱3：取余的符号
```python
# 公式：a = b * (a//b) + (a%b)
print(-5 % 3)     # 1（因为 -5 = 3*(-2) + 1）
print(5 % -3)     # -1（因为 5 = -3*(-2) + (-1)）
print(-5 % -3)    # -2（因为 -5 = -3*1 + (-2)）
```

## 陷阱4：布尔值是整数子类
```python
print(True + True)    # 2
print(True * 5)       # 5
print(False * 10)     # 0
print(42 + True)      # 43
```

## 陷阱5：大整数不会溢出
```python
# Python整数可以无限大
2 ** 1000        # 超大整数，不会溢出
10 ** 1000       # 照样能算，只是慢
```

## 陷阱6：除以0
```python
1 / 0            # ZeroDivisionError
1.0 / 0          # ZeroDivisionError（浮点也不行）
float('inf') * 0 # nan（特殊值运算没问题）
```

# 八、练习题

## 题1：结果是什么？
```python
print(0.1 + 0.2)
# 答案：0.30000000000000004
```

## 题2：结果是什么？
```python
print(5 // 2)
print(-5 // 2)
# 答案：2, -3
```

## 题3：结果是什么？
```python
print(-5 % 3)
# 答案：1
```

## 题4：结果是什么？
```python
print(True + False * 2)
# 答案：1（True=1，False=0，0*2=0，1+0=1）
```

## 题5：结果是什么？
```python
print(int(-3.7))
print(round(-3.7))
# 答案：-3, -4（int截断，round四舍五入）
```

## 题6：结果是什么？
```python
print(1e3)
print(1e-3)
# 答案：1000.0, 0.001
```
