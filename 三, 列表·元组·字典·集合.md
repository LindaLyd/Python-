# 一、列表（list）

## 1. 基本特性

| 特性 | 说明 |
|------|------|
| **有序** | 元素有固定顺序，支持索引 |
| **可变** | 可以增删改元素 |
| **可重复** | 允许重复元素 |
| **任意类型** | 可以混合存储不同类型 |

```python
# 创建列表
lst1 = [1, 2, 3]
lst2 = list("abc")        # ['a', 'b', 'c']
lst3 = [1, "hello", 3.14]  # 混合类型
lst4 = []                 # 空列表
```

## 2. 列表切片（高频考点）

```python
lst = [0, 1, 2, 3, 4, 5]

# 基本切片 [start:end]（包左不包右）
lst[2:4]      # [2, 3]
lst[:3]       # [0, 1, 2]（省略开始，从0开始）
lst[3:]       # [3, 4, 5]（省略结束，到最后）
lst[:]        # [0, 1, 2, 3, 4, 5]（复制整个列表）

# 步长切片 [start:end:step]
lst[::2]      # [0, 2, 4]（步长2）
lst[1::2]     # [1, 3, 5]（步长2，从索引1开始）
lst[::-1]     # [5, 4, 3, 2, 1, 0]（反转列表）

# 负索引切片
lst[-3:]      # [3, 4, 5]（最后3个）
lst[:-2]      # [0, 1, 2, 3]（去掉最后2个）
```

## 3. 列表推导式（必考）

```python
# 基本语法：[表达式 for 变量 in 可迭代对象 if 条件]

# 生成平方列表
squares = [x**2 for x in range(1, 6)]      # [1, 4, 9, 16, 25]

# 带条件筛选偶数
evens = [x for x in range(10) if x % 2 == 0]  # [0, 2, 4, 6, 8]

# 带条件判断
results = ["偶数" if x % 2 == 0 else "奇数" for x in range(5)]
# ['偶数', '奇数', '偶数', '奇数', '偶数']

# 嵌套循环
pairs = [(x, y) for x in [1,2] for y in [3,4]]
# [(1,3), (1,4), (2,3), (2,4)]

# 字符串处理
words = ["hello", "world", "python"]
upper_words = [w.upper() for w in words]  # ['HELLO', 'WORLD', 'PYTHON']
```

## 4. 列表常用方法

| 方法 | 作用 | 示例 |
|------|------|------|
| `append(x)` | 末尾添加元素 | `lst.append(4)` |
| `extend(iter)` | 末尾添加多个元素 | `lst.extend([4,5])` |
| `insert(i, x)` | 指定位置插入 | `lst.insert(0, 'a')` |
| `remove(x)` | 删除第一个匹配项 | `lst.remove(2)` |
| `pop(i)` | 删除并返回指定位置 | `lst.pop()`（末尾） |
| `index(x)` | 返回第一个匹配索引 | `lst.index(3)` |
| `count(x)` | 统计元素出现次数 | `lst.count(1)` |
| `sort()` | 原地排序 | `lst.sort()` |
| `reverse()` | 原地反转 | `lst.reverse()` |
| `clear()` | 清空列表 | `lst.clear()` |

```python
# 重要区别
# sort() vs sorted()
lst = [3, 1, 4, 1, 5]

lst.sort()        # 原地修改，返回None
print(lst)        # [1, 1, 3, 4, 5]

new_lst = sorted(lst)  # 返回新列表，原列表不变
print(new_lst)    # [1, 1, 3, 4, 5]

# reverse() vs [::-1]
lst.reverse()     # 原地反转
reversed_lst = lst[::-1]  # 返回新列表
```

# 二、元组（tuple）

## 1. 基本特性

| 特性 | 说明 |
|------|------|
| **有序** | 元素有固定顺序 |
| **不可变** | 创建后不能修改 |
| **可重复** | 允许重复元素 |
| **可哈希** | 可以作为字典键（元素也需不可变） |

```python
# 创建元组
t1 = (1, 2, 3)
t2 = tuple([1, 2, 3])    # (1, 2, 3)
t3 = 1, 2, 3             # (1, 2, 3)（不加括号也可以）
t4 = (1,)                # 单元素元组，必须加逗号！
t5 = ()                  # 空元组

# ❌ 错误
t6 = (1)                 # 这是整数1，不是元组！
```

## 2. 元组的不可变性

```python
t = (1, 2, 3)

# ❌ 以下操作会报错
t[0] = 10      # TypeError: 'tuple' object does not support item assignment
t.append(4)    # AttributeError
t.remove(2)    # AttributeError

# ✅ 可以访问
print(t[0])    # 1
print(t[1:])   # (2, 3)

# ✅ 可以重新赋值整个元组
t = (4, 5, 6)  # 这不是修改，是重新绑定
```

## 3. 元组的特殊场景

```python
# 函数返回多个值（自动打包为元组）
def get_user():
    return "张三", 25  # 实际返回 ("张三", 25)

name, age = get_user()  # 元组解包

# 元组解包
a, b, c = (1, 2, 3)
a, b = b, a  # 交换变量（本质是元组解包）

# 作为字典键
d = {}
d[(1, 2)] = "坐标"  # ✅ 元组可以作为键
# d[[1, 2]] = "坐标"  # ❌ 列表不能作为键
```

# 三、字典（dict）

## 1. 基本特性

| 特性 | 说明 |
|------|------|
| **无序**（Python 3.7+ 有序） | 键值对存储，3.7+按插入顺序 |
| **可变** | 可以增删改 |
| **键唯一** | 键不能重复 |
| **键必须可哈希** | 键必须是不可变类型 |
| **值任意** | 值可以是任何类型 |

```python
# 创建字典
d1 = {"name": "张三", "age": 25}
d2 = dict(name="张三", age=25)      # 关键字参数
d3 = dict([("name", "张三"), ("age", 25)])  # 键值对列表
d4 = {}                           # 空字典
d5 = {}.fromkeys(["a", "b"], 0)   # {'a': 0, 'b': 0}
```

## 2. 字典键值操作（高频考点）

| 操作 | 方法 | 示例 |
|------|------|------|
| **获取值** | `d[key]` | 键不存在报错 |
| **安全获取** | `d.get(key, default)` | 键不存在返回默认值 |
| **设置默认值** | `d.setdefault(key, default)` | 键不存在则设置 |
| **添加/修改** | `d[key] = value` | 键存在则修改，不存在则添加 |
| **更新** | `d.update(other_dict)` | 合并字典 |
| **删除** | `del d[key]` | 键不存在报错 |
| **安全删除** | `d.pop(key, default)` | 键不存在返回默认值 |
| **删除最后** | `d.popitem()` | 删除并返回键值对 |
| **清空** | `d.clear()` | 清空所有 |

```python
d = {"a": 1, "b": 2}

# ✅ 推荐：使用 get() 安全取值
print(d.get("a"))        # 1
print(d.get("c"))        # None（不报错）
print(d.get("c", 0))     # 0（指定默认值）

# setdefault：键不存在时设置
d.setdefault("c", 3)     # 返回3，d变成{"a":1,"b":2,"c":3}
d.setdefault("a", 100)   # 返回1（已有，不修改）

# pop：删除并返回值
value = d.pop("a")       # 返回1，d变成{"b":2,"c":3}
value = d.pop("x", "不存在")  # 返回"不存在"（不报错）

# 遍历字典
for key in d:                    # 遍历键
    print(key, d[key])

for key, value in d.items():     # 遍历键值对
    print(key, value)

for value in d.values():         # 遍历值
    print(value)
```

## 3. 字典合并（Python 3.9+）

```python
d1 = {"a": 1, "b": 2}
d2 = {"c": 3, "d": 4}

# 方法1：使用 | 运算符（Python 3.9+）
merged = d1 | d2        # {'a':1, 'b':2, 'c':3, 'd':4}

# 方法2：使用 update
d1.update(d2)           # d1被修改

# 方法3：解包
merged = {**d1, **d2}
```

## 4. 字典推导式

```python
# 基本语法：{key_expr: value_expr for item in iterable if condition}

# 创建平方字典
squares = {x: x**2 for x in range(5)}
# {0:0, 1:1, 2:4, 3:9, 4:16}

# 交换键值
d = {"a": 1, "b": 2, "c": 3}
reversed_d = {v: k for k, v in d.items()}  # {1:'a', 2:'b', 3:'c'}

# 带条件筛选
filtered = {k: v for k, v in d.items() if v > 1}  # {'b':2, 'c':3}
```

# 四、集合（set）

## 1. 基本特性

| 特性 | 说明 |
|------|------|
| **无序** | 元素没有顺序 |
| **可变** | 可以增删 |
| **元素唯一** | 自动去重 |
| **元素必须可哈希** | 元素必须是不可变类型 |

```python
# 创建集合
s1 = {1, 2, 3}
s2 = set([1, 2, 2, 3])    # {1, 2, 3}（自动去重）
s3 = set()                 # 空集合（不能用{}，那是空字典）

# ❌ 集合元素必须是不可变的
s = {[1, 2]}               # TypeError（列表不能作为元素）
s = {1, 2, (3, 4)}         # ✅ 元组可以
```

## 2. 集合运算（高频考点）

```python
a = {1, 2, 3, 4}
b = {3, 4, 5, 6}

# 并集 |
print(a | b)        # {1, 2, 3, 4, 5, 6}
print(a.union(b))   # 同上

# 交集 &
print(a & b)        # {3, 4}
print(a.intersection(b))  # 同上

# 差集 -
print(a - b)        # {1, 2}（在a不在b）
print(b - a)        # {5, 6}（在b不在a）
print(a.difference(b))  # 同上

# 对称差集 ^（并集减去交集）
print(a ^ b)        # {1, 2, 5, 6}
print(a.symmetric_difference(b))  # 同上

# 判断子集/超集
print(a.issubset(b))    # False（a是否是b的子集）
print(a.issuperset(b))  # False（a是否是b的超集）
```

## 3. 集合常用方法

| 方法 | 作用 | 示例 |
|------|------|------|
| `add(x)` | 添加元素 | `s.add(5)` |
| `remove(x)` | 删除元素，不存在报错 | `s.remove(5)` |
| `discard(x)` | 删除元素，不存在不报错 | `s.discard(5)` |
| `pop()` | 随机删除并返回 | `s.pop()` |
| `clear()` | 清空集合 | `s.clear()` |

## 4. 集合推导式

```python
# 基本语法：{expression for item in iterable if condition}

# 生成平方集合
squares = {x**2 for x in range(5)}  # {0, 1, 4, 9, 16}

# 去重并筛选
numbers = [1, 2, 2, 3, 4, 4, 5]
unique_evens = {x for x in numbers if x % 2 == 0}  # {2, 4}
```

# 六、经典陷阱题

## 陷阱1：列表作为默认参数

```python
# ❌ 错误
def add_item(item, lst=[]):
    lst.append(item)
    return lst

print(add_item(1))  # [1]
print(add_item(2))  # [1, 2] ← 不是 [2]！

# ✅ 正确
def add_item(item, lst=None):
    if lst is None:
        lst = []
    lst.append(item)
    return lst
```

## 陷阱2：遍历时修改列表

```python
# ❌ 错误
lst = [1, 2, 3, 4]
for i in lst:
    if i % 2 == 0:
        lst.remove(i)  # 遍历时修改，索引错乱
print(lst)  # [1, 3]（但结果不可预测）

# ✅ 正确
lst = [i for i in lst if i % 2 != 0]  # 创建新列表
```

## 陷阱3：列表复制

```python
a = [1, 2, 3]

# 浅复制
b = a[:]          # 或 a.copy() 或 list(a)
b = a.copy()      # 创建新列表

# 深复制（嵌套列表）
import copy
c = copy.deepcopy(a)
```

## 陷阱4：字典键必须是不可变

```python
# ✅ 可以
d = {(1, 2): "tuple"}      # 元组可以
d = {"abc": "string"}      # 字符串可以
d = {1: "int"}             # 整数可以

# ❌ 不可以
d = {[1, 2]: "list"}       # TypeError
d = {{"a":1}: "dict"}      # TypeError
```

## 陷阱5：集合自动去重

```python
# 常见应用
numbers = [1, 2, 2, 3, 3, 3, 4]
unique = list(set(numbers))  # [1, 2, 3, 4]（但顺序可能乱）

# 保持顺序（Python 3.7+）
unique = list(dict.fromkeys(numbers))  # [1, 2, 3, 4]
```

# 七、练习题

## 题1：列表切片
```python
lst = [0, 1, 2, 3, 4, 5]
print(lst[1:4])      # 答案：[1, 2, 3]
print(lst[-2:])      # 答案：[4, 5]
print(lst[::-2])     # 答案：[5, 3, 1]
```

## 题2：列表推导式
```python
[x**2 for x in range(1, 6) if x % 2 == 1]
# 答案：[1, 9, 25]（1,3,5的平方）
```

## 题3：字典操作
```python
d = {"a": 1, "b": 2}
print(d.get("c", 0))           # 答案：0
print(d.pop("b"))              # 答案：2
print(d)                       # 答案：{"a": 1}
```

## 题4：集合运算
```python
a = {1, 2, 3}
b = {2, 3, 4}
print(a | b)      # 答案：{1, 2, 3, 4}
print(a & b)      # 答案：{2, 3}
print(a - b)      # 答案：{1}
```

## 题5：去重
```python
nums = [1, 2, 2, 3, 3, 3, 4]
unique = list(set(nums))
# 答案可能是：[1, 2, 3, 4]（顺序不保证）
```
