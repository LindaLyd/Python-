# 一、基础运算符分类

## 1. 算术运算符（数学计算）

| 运算符 | 含义 | 示例 | 结果 |
|--------|------|------|------|
| `+` | 加法 | `5 + 3` | `8` |
| `-` | 减法 | `5 - 3` | `2` |
| `*` | 乘法 | `5 * 3` | `15` |
| `/` | 除法（浮点数） | `5 / 2` | `2.5` |
| `//` | 整除（向下取整） | `5 // 2` | `2` |
| `%` | 取余（模运算） | `5 % 2` | `1` |
| `**` | 幂运算 | `5 ** 2` | `25` |

**特殊注意：**
```python
print(-5 // 2)   # -3（向下取整，-2.5 向下是 -3）
print(5 // -2)   # -3
print(-5 % 2)    # 1（余数规律：a = b * (a//b) + (a%b)）
```

## 2. 比较运算符（返回boolean）

| 运算符 | 含义 | 示例 | 结果 |
|--------|------|------|------|
| `==` | 等于 | `5 == 3` | `False` |
| `!=` | 不等于 | `5 != 3` | `True` |
| `>` | 大于 | `5 > 3` | `True` |
| `<` | 小于 | `5 < 3` | `False` |
| `>=` | 大于等于 | `5 >= 5` | `True` |
| `<=` | 小于等于 | `5 <= 3` | `False` |
| `is` | 同一对象 | `a is b` | 地址相同 |
| `is not` | 不同对象 | `a is not b` | 地址不同 |
| `in` | 成员属于 | `'a' in 'abc'` | `True` |
| `not in` | 成员不属于 | `'d' not in 'abc'` | `True` |

**`==` vs `is` 核心区别：**
```python
a = [1,2,3]
b = [1,2,3]
c = a

print(a == b)   # True（值相等）
print(a is b)   # False（不同对象，不同内存地址）
print(a is c)   # True（同一对象，同一内存地址）
```

## 3. 赋值运算符

| 运算符 | 示例 | 等价于 |
|--------|------|--------|
| `=` | `x = 5` | 赋值 |
| `+=` | `x += 3` | `x = x + 3` |
| `-=` | `x -= 3` | `x = x - 3` |
| `*=` | `x *= 3` | `x = x * 3` |
| `/=` | `x /= 3` | `x = x / 3` |
| `//=` | `x //= 3` | `x = x // 3` |
| `%=` | `x %= 3` | `x = x % 3` |
| `**=` | `x **= 3` | `x = x ** 3` |

**特殊赋值技巧：**
```python
# 多重赋值
a, b = 1, 2
a, b = b, a      # 交换变量，不用临时变量

# 链式赋值
x = y = z = 0    # x、y、z 都是 0
```

## 4. 逻辑运算符

| 运算符 | 含义 | 示例 | 结果 |
|--------|------|------|------|
| `and` | 与（都真才真） | `True and False` | `False` |
| `or` | 或（一真即真） | `True or False` | `True` |
| `not` | 非（取反） | `not True` | `False` |

**短路原则（重要考点）：**
```python
# and：第一个为 False，就不算后面的
def return_false():
    print("执行了")
    return False

def return_true():
    print("执行了")
    return True

# 短路现象
result = False and return_true()  # return_true() 不执行！因为第一个已经是 False
result = True or return_false()    # return_false() 不执行！因为第一个已经是 True
```

## 5. 位运算符（二进制操作）

| 运算符 | 含义 | 示例 | 二进制 |
|--------|------|------|--------|
| `&` | 按位与 | `5 & 3` | `101 & 011 = 001` (1) |
| `|` | 按位或 | `5 | 3` | `101 | 011 = 111` (7) |
| `^` | 按位异或 | `5 ^ 3` | `101 ^ 011 = 110` (6) |
| `~` | 按位取反 | `~5` | `...11111010` (-6) |
| `<<` | 左移 | `5 << 1` | `1010` (10) |
| `>>` | 右移 | `5 >> 1` | `10` (2) |

## 6. 成员/身份运算符

| 运算符 | 含义 | 示例 |
|--------|------|------|
| `in` | 在序列中 | `'a' in 'abc'` |
| `not in` | 不在序列中 | `'d' not in 'abc'` |
| `is` | 同一对象 | `a is b` |
| `is not` | 不同对象 | `a is not b` |

# 二、运算符优先级（必考重点）

## 完整优先级表（从高到低）

| 优先级 | 类别 | 运算符 | 结合性 |
|--------|------|--------|--------|
| **最高** | 索引/切片 | `[]`、`[:]` | 左 |
| | 幂运算 | `**` | **右**（唯一右结合） |
| | 正负号 | `+x`、`-x`、`~x` | 左 |
| | 算术乘除 | `*`、`/`、`//`、`%` | 左 |
| | 算术加减 | `+`、`-` | 左 |
| | 移位 | `<<`、`>>` | 左 |
| | 按位与 | `&` | 左 |
| | 按位异或 | `^` | 左 |
| | 按位或 | `|` | 左 |
| | 比较 | `in`、`not in`、`is`、`is not`、`<`、`<=`、`>`、`>=`、`!=`、`==` | 左 |
| | 逻辑非 | `not x` | 左 |
| | 逻辑与 | `and` | 左 |
| **最低** | 逻辑或 | `or` | 左 |

## 简化记忆版（考试够用）

```
最高：**（幂运算）
     +x、-x、~x（一元运算符）
     *、/、//、%（乘除取余）
     +、-（加减）
     <<、>>（移位）
     &（按位与）
     ^（按位异或）
     |（按位或）
     ==、!=、>、<、>=、<=、is、is not、in、not in（比较）
     not（逻辑非）
     and（逻辑与）
最低：or（逻辑或）
```

**口诀：** **"幂运算最高，乘除加减次，移位按位再比较，最后逻辑非与或"**

# 三、经典陷阱题（必考）

## 陷阱1：算术 vs 比较 vs 逻辑
```python
# 题目：2 + 3 * 4 > 5 and True
# 执行顺序：
# 1. 3 * 4 = 12
# 2. 2 + 12 = 14
# 3. 14 > 5 = True
# 4. True and True = True
print(2 + 3 * 4 > 5 and True)  # True
```

## 陷阱2：链式比较
```python
# 题目：5 > 3 == True
# 等价于：(5 > 3) and (3 == True)
# 3 == True 相当于 3 == 1，结果是 False
print(5 > 3 == True)  # False（经典坑题！）
```

## 陷阱3：赋值 vs 比较
```python
x = 5
y = 10

# 错误的写法（常见笔误）
if x = y:  # ❌ SyntaxError，赋值不能用在if条件里
    print("相等")

# 正确写法
if x == y:  # ✅ 比较
    print("相等")
```

## 陷阱4：逻辑运算符返回值
```python
# and/or 返回的是最后一个被计算的值，不一定是布尔值
print(3 and 5)      # 5（3为真，返回5）
print(0 and 5)      # 0（0为假，返回0）
print(3 or 5)       # 3（3为真，返回3）
print(0 or 5)       # 5（0为假，返回5）

# 实际应用：设置默认值
name = input_name or "默认值"  # 如果input_name为空，用"默认值"
```

## 陷阱5：幂运算的右结合
```python
# 幂运算是右结合
print(2 ** 3 ** 2)  # 2 ** (3 ** 2) = 2 ** 9 = 512
# 不是 (2 ** 3) ** 2 = 8 ** 2 = 64
```

## 陷阱6：取余和整除的符号
```python
print(-7 // 3)   # -3（向下取整，-2.33向下是-3）
print(7 // -3)   # -3
print(-7 % 3)    # 2（满足公式：-7 = 3*(-3) + 2）
```

# 五、实战练习题

## 题1：说出结果
```python
print(2 + 3 * 4)
# 答案：14（先乘后加）
```

## 题2：说出结果
```python
print(5 > 4 > 3)
# 答案：True（等价于 5>4 and 4>3）
```

## 题3：说出结果
```python
print(2 ** 3 ** 2)
# 答案：512（右结合：2 ** (3 ** 2)）
```

## 题4：说出结果
```python
print(3 and 5 or 0)
# 答案：5（先算 3 and 5 = 5，再算 5 or 0 = 5）
```

## 题5：说出结果
```python
x = 5
y = 10
print(x == y and 3 > 2 or not False)
# 答案：True
# 步骤：x==y=False → False and True=False → False or not False=False or True=True
```
