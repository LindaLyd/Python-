# 二、变量作用域

## 1. 作用域类型（LEGB规则）

| 作用域 | 英文 | 说明 | 示例 |
|--------|------|------|------|
| **L**ocal | 局部作用域 | 函数内部定义的变量 | `def func(): x = 1` |
| **E**nclosing | 外层嵌套作用域 | 外层函数的变量 | 闭包中的外层函数 |
| **G**lobal | 全局作用域 | 模块顶层定义的变量 | 文件最外层 `x = 1` |
| **B**uilt-in | 内置作用域 | Python内置名字 | `print`, `len` |

**查找顺序：** L → E → G → B （由内向外找）

## 2. 全局变量 vs 局部变量

```python
# 全局变量
x = 100

def func():
    # 局部变量
    y = 200
    print(x)  # 可以访问全局变量（读取）
    print(y)  # 可以访问局部变量

func()
print(x)      # ✅ 可以
print(y)      # ❌ NameError: y 只在函数内有效
```

## 3. 修改变量的关键：`global` 和 `nonlocal`

### global：修改全局变量
```python
x = 10

def change_global():
    global x    # 声明 x 是全局的
    x = 20      # 修改全局变量

change_global()
print(x)  # 20 ✅ 全局变量被修改了

# 如果不加 global
def try_change():
    x = 30      # ❌ 这是创建了一个新的局部变量，不是修改全局的

try_change()
print(x)  # 20（没变）
```

### nonlocal：修改外层函数的变量
```python
def outer():
    x = 10
    
    def inner():
        nonlocal x  # 声明 x 来自外层函数
        x = 20      # 修改 outer 里的 x
    
    inner()
    print(x)  # 20 ✅ 外层变量被修改了

# 如果不加 nonlocal
def outer2():
    x = 10
    def inner2():
        x = 20      # ❌ 创建新的局部变量
    inner2()
    print(x)  # 10（没变）
```

## 4. 作用域经典陷阱题

### 陷阱1：函数内部修改全局变量
```python
count = 0

def increment():
    count += 1   # ❌ UnboundLocalError
    # 因为 Python 看到赋值，就认为 count 是局部变量，但还没定义

# 正确写法
def increment():
    global count
    count += 1
```

### 陷阱2：可变对象的特殊情况
```python
my_list = [1,2,3]

def add_item():
    my_list.append(4)  # ✅ 不需要 global！修改列表内容不是重新赋值

add_item()
print(my_list)  # [1,2,3,4]

def reassign():
    my_list = [5,6,7]  # ❌ 这是创建新的局部变量，不是修改全局的

reassign()
print(my_list)  # 还是 [1,2,3,4]
```

**关键区别：**
- **修改内容**（`.append()`）→ 不需要 `global`
- **重新赋值**（`=`）→ 需要 `global`

### 陷阱3：作用域嵌套查找
```python
x = 1

def outer():
    x = 2
    def inner():
        x = 3
        print(x)  # 3（先找局部）
    inner()
    print(x)      # 2（外层变量没变）

outer()
print(x)          # 1（全局没变）

# 输出：3 2 1
```

## 5. 考试速记表

| 场景 | 关键字 | 作用 |
|------|--------|------|
| 在函数内修改全局变量 | `global` | 声明变量来自全局 |
| 在嵌套函数内修改外层变量 | `nonlocal` | 声明变量来自外层函数 |
| 只是读取全局变量 | 不需要 | 直接读就行 |
| 只是读取外层变量 | 不需要 | 直接读就行 |

## 6. 口诀记忆

```
变量命名要记牢：
字母数字下划线，数字千万别打头
关键字名不能用，大小写要分清楚
内置函数别覆盖，下划线前是私有

作用域里找变量：
LEGB 往外找，局部没有找外层
外层没有找全局，全局没有再找内置
修改全局用 global，修改外层 nonlocal
可变对象改内容，不用声明也能行
```
