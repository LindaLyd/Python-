## 【字符串操作】系统性知识点

# 一、字符串基础特性

## 1. 创建字符串

```python
# 单引号
s1 = 'Hello'

# 双引号（和单引号没区别）
s2 = "World"

# 三引号（多行字符串）
s3 = '''第一行
第二行
第三行'''

s4 = """也可以这样
跨越多行"""
```

## 2. 字符串是不可变的

```python
s = "Hello"
s[0] = "h"  # ❌ TypeError: 'str' object does not support item assignment

# 正确方式：创建新字符串
s = "h" + s[1:]  # "hello"
```

## 3. 转义字符

| 转义符 | 含义 |
|--------|------|
| `\n` | 换行 |
| `\t` | 制表符（Tab） |
| `\\` | 反斜杠本身 |
| `\'` | 单引号 |
| `\"` | 双引号 |
| `\r` | 回车 |

```python
print("Hello\nWorld")   # Hello\nWorld（换行）
print("C:\\Users\\name") # C:\Users\name
print('It\'s ok')        # It's ok

# 原始字符串（忽略转义）
print(r"C:\Users\name")  # C:\Users\name（\U不是转义）
```

# 二、字符串索引和切片

## 1. 索引（正负索引）

```python
s = "Python"

# 正索引（从左到右，从0开始）
print(s[0])    # 'P'
print(s[1])    # 'y'
print(s[5])    # 'n'

# 负索引（从右到左，从-1开始）
print(s[-1])   # 'n'
print(s[-2])   # 'o'
print(s[-6])   # 'P'
```

## 2. 切片 `[start:end:step]`

```python
s = "Python"

# 基本切片（包左不包右）
s[0:2]    # 'Py'（索引0到1）
s[2:4]    # 'th'
s[:4]     # 'Pyth'（省略开始，从0开始）
s[2:]     # 'thon'（省略结束，到末尾）
s[:]      # 'Python'（复制整个字符串）

# 步长切片
s[::2]    # 'Pto'（步长2，取索引0,2,4）
s[1::2]   # 'yhn'（步长2，取索引1,3,5）
s[::-1]   # 'nohtyP'（步长-1，反转字符串）

# 带步长的完整形式
s[1:5:2]  # 'yh'（索引1到4，步长2）
```

## 3. 切片记忆口诀

```
切片格式 [开始:结束:步长]
包左不包右，负号倒着走
开始省略取开头，结束省略到最后
步长负数就反转
```

# 三、字符串常用方法

## 1. 查找与统计

| 方法 | 作用 | 示例 |
|------|------|------|
| `find(sub)` | 返回首次出现索引，找不到返回-1 | `"hello".find('l')` → `2` |
| `index(sub)` | 返回首次出现索引，找不到报错 | `"hello".index('l')` → `2` |
| `rfind(sub)` | 从右查找，返回索引 | `"hello".rfind('l')` → `3` |
| `count(sub)` | 统计子串出现次数 | `"hello".count('l')` → `2` |
| `startswith(prefix)` | 是否以某前缀开头 | `"hello".startswith('he')` → `True` |
| `endswith(suffix)` | 是否以某后缀结尾 | `"hello".endswith('lo')` → `True` |

```python
text = "Hello World"

print(text.find('o'))      # 4
print(text.find('x'))      # -1（找不到）
print(text.index('o'))     # 4
# print(text.index('x'))   # ValueError
print(text.count('l'))     # 3
print(text.startswith('He')) # True
```

## 2. 大小写转换

| 方法 | 作用 | 示例 |
|------|------|------|
| `upper()` | 全转大写 | `"hello".upper()` → `"HELLO"` |
| `lower()` | 全转小写 | `"HELLO".lower()` → `"hello"` |
| `capitalize()` | 首字母大写 | `"hello".capitalize()` → `"Hello"` |
| `title()` | 每个单词首字母大写 | `"hello world".title()` → `"Hello World"` |
| `swapcase()` | 大小写互换 | `"Hello".swapcase()` → `"hELLO"` |

## 3. 去除空白

| 方法 | 作用 | 示例 |
|------|------|------|
| `strip()` | 去除两端空白 | `"  hello  ".strip()` → `"hello"` |
| `lstrip()` | 去除左侧空白 | `"  hello".lstrip()` → `"hello"` |
| `rstrip()` | 去除右侧空白 | `"hello  ".rstrip()` → `"hello"` |

```python
s = "  hello world  \n"
print(s.strip())      # "hello world"
print(s.lstrip())     # "hello world  \n"
print(s.rstrip())     # "  hello world"
```

## 4. 拆分与拼接（高频考点）

| 方法 | 作用 | 示例 |
|------|------|------|
| `split(sep)` | 按分隔符拆分为列表 | `"a,b,c".split(',')` → `['a','b','c']` |
| `split()` | 按空白拆分为列表 | `"a b c".split()` → `['a','b','c']` |
| `join(iterable)` | 用字符串拼接可迭代对象 | `','.join(['a','b','c'])` → `"a,b,c"` |

```python
# split 拆分成列表
text = "apple,banana,orange"
fruits = text.split(',')
print(fruits)  # ['apple', 'banana', 'orange']

# split 默认按空白拆分
words = "Hello  World   Python".split()
print(words)  # ['Hello', 'World', 'Python']

# join 拼接列表为字符串
colors = ['red', 'green', 'blue']
result = '-'.join(colors)
print(result)  # "red-green-blue"

# 常用技巧：拆分后拼接
s = "a,b,c,d"
print(','.join(s.split(',')))  # "a,b,c,d"（重新组合）
```

## 5. 替换与填充

| 方法 | 作用 | 示例 |
|------|------|------|
| `replace(old, new)` | 替换子串 | `"hello".replace('l','x')` → `"hexxo"` |
| `replace(old, new, count)` | 替换指定次数 | `"hello".replace('l','x',1)` → `"hexlo"` |
| `ljust(width, fillchar)` | 左对齐填充 | `"hi".ljust(5,'*')` → `"hi***"` |
| `rjust(width, fillchar)` | 右对齐填充 | `"hi".rjust(5,'*')` → `"***hi"` |
| `center(width, fillchar)` | 居中对齐 | `"hi".center(5,'*')` → `"*hi**"` |
| `zfill(width)` | 用0填充 | `"42".zfill(5)` → `"00042"` |

## 6. 判断类型

| 方法 | 作用 | 示例 |
|------|------|------|
| `isdigit()` | 是否全是数字 | `"123".isdigit()` → `True` |
| `isalpha()` | 是否全是字母 | `"abc".isalpha()` → `True` |
| `isalnum()` | 是否字母或数字 | `"abc123".isalnum()` → `True` |
| `isspace()` | 是否全是空白 | `"   ".isspace()` → `True` |
| `islower()` | 是否全小写 | `"hello".islower()` → `True` |
| `isupper()` | 是否全大写 | `"HELLO".isupper()` → `True` |

# 四、字符串格式化

## 1. f-string（Python 3.6+，推荐）

```python
name = "张三"
age = 25
score = 85.5

# 基本用法
print(f"姓名：{name}，年龄：{age}")  # 姓名：张三，年龄：25

# 表达式
print(f"明年年龄：{age + 1}")        # 明年年龄：26

# 格式化数字
print(f"分数：{score:.1f}")          # 分数：85.5
print(f"百分比：{score:.0%}")        # 百分比：86%

# 对齐填充
print(f"{name:<10}")  # 左对齐，宽度10
print(f"{name:>10}")  # 右对齐
print(f"{name:^10}")  # 居中对齐
print(f"{name:*^10}") # 居中对齐，*填充

# 调用方法
print(f"大写：{name.upper()}")       # 大写：张三（但中文没变化）
```

## 2. format() 方法

```python
# 位置参数
"{} {}".format("Hello", "World")     # "Hello World"
"{1} {0}".format("Hello", "World")   # "World Hello"

# 关键字参数
"{name}今年{age}岁".format(name="张三", age=25)

# 格式化数字
"{:.2f}".format(3.14159)             # "3.14"
```

## 3. % 格式化（老式）

```python
"%s今年%d岁" % ("张三", 25)           # "张三今年25岁"
"分数：%.1f" % 85.5                  # "分数：85.5"
```

# 五、字符串其他操作

## 1. 成员运算符 `in` 和 `not in`

```python
s = "Hello World"

print("Hello" in s)      # True
print("hi" in s)         # False
print("World" not in s)  # False
```

## 2. 字符串连接与重复

```python
# 连接
s1 = "Hello" + " " + "World"  # "Hello World"

# 重复
s2 = "Hi" * 3                  # "HiHiHi"

# 多行字符串自动连接
s3 = ("Hello"
      "World")                 # "HelloWorld"
```

## 3. 获取字符串长度

```python
s = "Python"
print(len(s))      # 6
```

## 4. 遍历字符串

```python
s = "Python"

# 遍历字符
for char in s:
    print(char)

# 遍历索引
for i in range(len(s)):
    print(s[i])

# 同时取索引和值
for i, char in enumerate(s):
    print(i, char)
```

# 六、经典陷阱题

## 陷阱1：字符串不可变

```python
s = "Hello"
s[0] = "h"  # ❌ TypeError

# ✅ 正确方式
s = "h" + s[1:]  # "hello"
```

## 陷阱2：split 和 join 的配合

```python
# 题目：把字符串按空格拆开，再用逗号拼回去
s = "a b c"
# ✅ 正确
result = ','.join(s.split())
print(result)  # "a,b,c"

# ❌ 错误（join需要可迭代对象）
','.join(s)    # "a,b,c"？不，是 "a, ,b, ,c"（字符逐个拆）
```

## 陷阱3：切片不修改原字符串

```python
s = "Python"
s[2:4] = "xx"   # ❌ 切片不能赋值，字符串不可变

# ✅ 正确
s = s[:2] + "xx" + s[4:]  # "Pyxxon"
```

## 陷阱4：find vs index

```python
s = "hello"
print(s.find('x'))   # -1
print(s.index('x'))  # ValueError（会报错）
```

# 七、练习题

## 题1：切片结果
```python
s = "Python"
print(s[1:4])     # 答案：yth
print(s[-3:])     # 答案：hon
print(s[::-1])    # 答案：nohtyP
```

## 题2：split/join
```python
s = "one,two,three"
parts = s.split(',')
result = '-'.join(parts)
print(result)
# 答案：one-two-three
```

## 题3：替换
```python
s = "hello world"
print(s.replace('l', 'x'))      # 答案：hexxo worxd
print(s.replace('l', 'x', 2))   # 答案：hexxo world
```

## 题4：去除空白
```python
s = "  hello  "
print(s.strip())      # 答案：hello
print(s.lstrip())     # 答案：hello__
print(s.rstrip())     # 答案：__hello
```

## 题5：f-string
```python
name = "张三"
score = 85.567
print(f"{name}得分{score:.1f}分")
# 答案：张三得分85.6分
```

## 题6：判断类型
```python
print("123".isdigit())      # True
print("123".isalpha())      # False
print("abc123".isalnum())   # True
print("   ".isspace())      # True
```
