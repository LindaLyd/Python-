## 代码结构规范（PEP8）

---

# 一、缩进规范（PEP8核心）

## 1. 缩进规则

| 规则 | 正确 | 错误 |
|------|------|------|
| **使用4个空格**（不能用Tab或混用） | `····if True:` | `\tif True:` 或 空格+Tab |
| **同一代码块缩进一致** | 所有行都是4空格 | 有的4空格，有的3空格 |

```python
# ✅ 正确
def func():
····if True:
········print("Hello")
····else:
········print("World")

# ❌ 错误（缩进不一致）
def func():
····if True:
······print("Hello")  # 只有2空格
```

## 2. 换行缩进（续行）

```python
# 方式1：与括号对齐
foo = long_function_name(var_one, var_two,
                         var_three, var_four)

# 方式2：悬挂缩进（额外4空格）
foo = long_function_name(
····var_one, var_two,
····var_three, var_four)

# 方式3：运算符后换行（运算符放前面更好）
income = (gross_wages
          + taxable_interest
          - deductions)
```

# 二、空行规范

## 1. 空行数量

| 位置 | 空行数 | 示例 |
|------|--------|------|
| **顶级函数/类之间** | 2个空行 | `def func1(): ...` ⏎⏎ `def func2():` |
| **类内方法之间** | 1个空行 | `class MyClass:` ⏎ `····def method1():` ⏎ `····def method2():` |
| **逻辑代码块之间** | 1个空行（可选） | 相关代码组之间加空行 |

```python
import os
import sys  # 导入后空两行

⏎
⏎
class MyClass:
····def __init__(self):
········pass
····⏎
····def method1(self):
········pass
⏎
⏎
def main():
····# 处理输入
····data = get_input()
····⏎
····# 处理数据
····result = process(data)
····⏎
····# 输出结果
····print(result)
```

# 三、导入规范

## 1. 导入顺序（三大组，组间空行）

```python
# 第一组：标准库
import os
import sys
from datetime import datetime
⏎
# 第二组：第三方库
import requests
import pandas as pd
from flask import Flask
⏎
# 第三组：本地库
from mypackage import mymodule
from . import helper
```

## 2. 每组内部排序

- **按字母顺序**排列
- `import ...` 排在 `from ... import ...` 前面

```python
# ✅ 正确
import json
import os
import sys
from datetime import datetime
from pathlib import Path

# ❌ 错误
import sys
import os
from pathlib import Path
import json
```

## 3. 单行 vs 多行

```python
# ✅ 推荐（每行一个）
import os
import sys
import json

# ✅ 可以（同模块多个导入）
from os import path, getcwd

# ❌ 不推荐（一行多个）
import os, sys, json
```

# 四、命名规范

| 类型 | 规范 | 示例 |
|------|------|------|
| **变量名** | 小写+下划线 | `user_name`, `total_count` |
| **常量** | 全大写+下划线 | `MAX_SIZE`, `PI` |
| **函数名** | 小写+下划线 | `get_user()`, `calculate_total()` |
| **类名** | 驼峰（首字母大写） | `UserInfo`, `CarModel` |
| **方法名** | 小写+下划线 | `get_name()`, `set_age()` |
| **私有属性/方法** | 前加单下划线 | `_internal_use`, `_private_method()` |
| **魔术方法** | 前后双下划线 | `__init__`, `__str__` |
| **避免的名字** | 不要用内置函数名 | `list`, `str`, `dict`, `len` |

# 五、注释规范

## 1. 行内注释

```python
x = x + 1  # 边界情况加1（注释前至少2空格）
```

## 2. 块注释

```python
# 这是块注释
# 每行以 # 加一个空格开头
# 说明下面代码的作用
def complex_function():
    pass
```

## 3. 文档字符串（Docstring）

```python
def add(a, b):
    """返回两个数的和。"""
    return a + b

class Person:
    """表示一个人的类。
    
    属性：
        name: 姓名
        age: 年龄
    """
    def __init__(self, name, age):
        pass
```

# 六、表达式和语句规范

## 1. 空格使用

```python
# ✅ 正确
spam(ham[1], {eggs: 2})  # 函数调用括号间无空格
x = 1                     # 赋值运算符两边空格
y = x + 1                 # 二元运算符两边空格
list[1] = 2               # 索引无空格

# ❌ 错误
spam( ham[ 1 ], { eggs: 2 } )
x=1
y=x+1
list[ 1 ] = 2
```

## 2. 行长度

- **每行最多79字符**（PEP8建议）
- 文档字符串/注释72字符

## 3. 多余空格

```python
# ❌ 多余空格
if x == 5 :        # 冒号前不要空格
print (x)          # 括号前不要空格
x = ( 1 + 2 ) * 3  # 括号内不要空格

# ✅ 正确
if x == 5:
print(x)
x = (1 + 2) * 3
```

# 七、比较规范

```python
# ✅ 推荐
if not my_list:           # 判断空列表
if my_list:               # 判断非空
if x is None:             # 判断None
if x is not None:         # 判断非None
if isinstance(x, int):    # 判断类型

# ❌ 不推荐
if len(my_list) == 0:     # 用 if not my_list
if my_list != []:         # 用 if my_list
if x == None:             # 用 if x is None
if type(x) == int:        # 用 isinstance
```
