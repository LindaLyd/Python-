# 一、条件判断（if-elif-else）

## 1. 基本语法

```python
# 单分支
if 条件:
    代码块

# 双分支
if 条件:
    代码块1
else:
    代码块2

# 多分支
if 条件1:
    代码块1
elif 条件2:
    代码块2
elif 条件3:
    代码块3
else:
    代码块4
```

## 2. 核心要点

| 要点 | 说明 | 示例 |
|------|------|------|
| **冒号** | 条件后面必须加 `:` | `if x > 5:` |
| **缩进** | 同一代码块缩进必须一致 | 通常用4个空格 |
| **elif** | 可以有0个或多个 | `if...elif...elif...` |
| **else** | 可选，最后出现 | 前面条件都不满足时执行 |

## 3. 条件表达式（三元运算符）

```python
# 语法：值1 if 条件 else 值2
age = 20
status = "成年" if age >= 18 else "未成年"
print(status)  # 成年

# 等价于
if age >= 18:
    status = "成年"
else:
    status = "未成年"
```

## 4. 真值判断（重要考点）

Python中哪些是**假值（False）**：

```python
# 以下6种情况在条件判断中相当于 False
bool(False)      # False
bool(None)       # False
bool(0)          # False
bool(0.0)        # False
bool('')         # 空字符串
bool([])         # 空列表
bool(())         # 空元组
bool({})         # 空字典
bool(set())      # 空集合

# 其他所有值都是 True
```

**常见简写技巧：**
```python
# 不推荐
if len(my_list) > 0:
    print("列表不为空")

# 推荐（直接判断列表本身）
if my_list:  # 非空列表为 True
    print("列表不为空")

# 不推荐
if x == True:
    print("x为真")

# 推荐
if x:
    print("x为真")
```

## 5. 常见陷阱

### 陷阱1：`==` 误写成 `=`
```python
x = 5
if x = 5:  # ❌ SyntaxError，赋值不能用在条件里
    print("相等")

# 正确
if x == 5:  # ✅
    print("相等")
```

### 陷阱2：空序列判断
```python
my_list = []

# ❌ 错误方式
if my_list == []:  # 可以，但不够Pythonic

# ✅ Pythonic方式
if not my_list:  # 空列表为False
    print("列表为空")
```

### 陷阱3：链式比较
```python
x = 5
# 数学写法可以直接用
if 3 < x < 7:  # ✅ 等价于 3 < x and x < 7
    print("x在3和7之间")
```

# 二、循环控制

## 1. for 循环

### 基本语法
```python
# 遍历序列
for 变量 in 可迭代对象:
    代码块

# 示例
for i in [1,2,3]:
    print(i)

for char in "Python":
    print(char)
```

### range() 函数（常用）
```python
range(stop)        # 0 到 stop-1
range(start, stop) # start 到 stop-1
range(start, stop, step) # 带步长

# 示例
for i in range(5):        # 0,1,2,3,4
for i in range(2, 5):     # 2,3,4
for i in range(1, 10, 2): # 1,3,5,7,9
for i in range(5, 0, -1): # 5,4,3,2,1（倒序）
```

### enumerate() 同时取索引和值
```python
fruits = ['苹果', '香蕉', '橙子']

# 不推荐
for i in range(len(fruits)):
    print(i, fruits[i])

# 推荐
for i, fruit in enumerate(fruits):
    print(i, fruit)  # 0 苹果, 1 香蕉, 2 橙子

# 指定起始索引
for i, fruit in enumerate(fruits, start=1):
    print(i, fruit)  # 1 苹果, 2 香蕉, 3 橙子
```

## 2. while 循环

### 基本语法
```python
while 条件:
    代码块
    # 记得更新条件，否则死循环

# 示例
count = 0
while count < 5:
    print(count)
    count += 1
```

### while-else（了解）
```python
count = 0
while count < 3:
    print(count)
    count += 1
else:
    print("循环正常结束")  # 循环条件为False时执行

# 如果循环被break，else不执行
count = 0
while count < 3:
    if count == 1:
        break
    print(count)
    count += 1
else:
    print("不会执行")  # 因为break了
```

## 3. 循环控制语句

| 语句 | 作用 | 示例 |
|------|------|------|
| `break` | 跳出整个循环 | `if x == 5: break` |
| `continue` | 跳过本次循环，继续下一次 | `if x % 2 == 0: continue` |
| `pass` | 占位符，什么都不做 | `if True: pass` |

### break 示例
```python
for i in range(10):
    if i == 5:
        break  # 到5就停止
    print(i)   # 输出：0,1,2,3,4
```

### continue 示例
```python
for i in range(5):
    if i == 2:
        continue  # 跳过2
    print(i)      # 输出：0,1,3,4
```

### pass 示例（常用于占位）
```python
if condition:
    pass  # 还没想好写什么，先占位
else:
    print("执行其他代码")

# 定义空函数/空类
def not_implemented_yet():
    pass

class MyClass:
    pass
```

## 4. for-else（重点考点）

```python
# 语法：for循环正常结束（没被break）时执行else
for item in sequence:
    if condition:
        break
else:
    # 循环中没有执行break才会执行这里
    print("循环完整执行完毕")

# 实际应用：查找元素
numbers = [1,3,5,7,9]
search = 4

for n in numbers:
    if n == search:
        print(f"找到{search}")
        break
else:
    print(f"没找到{search}")  # 执行这一行
```

# 三、循环进阶技巧

## 1. 列表推导式（简化for循环）

```python
# 传统方式
squares = []
for i in range(1, 6):
    squares.append(i**2)  # [1,4,9,16,25]

# 推导式
squares = [i**2 for i in range(1, 6)]

# 带条件的推导式
evens = [i for i in range(10) if i % 2 == 0]  # [0,2,4,6,8]

# 嵌套循环
pairs = [(x, y) for x in [1,2] for y in [3,4]]
# [(1,3), (1,4), (2,3), (2,4)]
```

## 2. zip() 并行迭代

```python
names = ['张三', '李四', '王五']
ages = [25, 30, 35]
scores = [85, 92, 78]

# 并行遍历多个列表
for name, age in zip(names, ages):
    print(f"{name}今年{age}岁")

# 遍历三个列表
for name, age, score in zip(names, ages, scores):
    print(f"{name}: {age}岁, {score}分")

# 如果列表长度不同，zip以最短的为准
```

## 3. 循环中的 else 总结

| 类型 | else 执行时机 | 不执行时机 |
|------|--------------|------------|
| for-else | 循环正常结束（没break） | 循环被break |
| while-else | 循环条件为False结束 | 循环被break |
| if-else | if条件为False | if条件为True |

# 四、经典陷阱题

## 陷阱1：for循环修改列表
```python
# 想在遍历时删除所有偶数
numbers = [1,2,3,4,5,6]

# ❌ 错误方式（索引会乱）
for i, n in enumerate(numbers):
    if n % 2 == 0:
        del numbers[i]  # 删除后索引错位

# ✅ 正确方式1：创建新列表
numbers = [n for n in numbers if n % 2 != 0]

# ✅ 正确方式2：从后往前遍历
for i in range(len(numbers)-1, -1, -1):
    if numbers[i] % 2 == 0:
        del numbers[i]
```

## 陷阱2：循环变量泄漏
```python
for i in range(5):
    pass
print(i)  # 4（在Python中，循环变量会保留到最后的值）

# 但在列表推导式中不会泄漏
[x for x in range(5)]
print(x)  # NameError（x不存在）
```

## 陷阱3：while循环死循环
```python
# ❌ 忘记更新条件
i = 0
while i < 5:
    print(i)
    # 没有 i += 1，死循环！

# ✅ 正确
i = 0
while i < 5:
    print(i)
    i += 1
```

## 陷阱4：break只跳出当前循环
```python
for i in range(3):
    print(f"外层{i}")
    for j in range(3):
        if j == 1:
            break  # 只跳出内层循环
        print(f"  内层{j}")
# 输出：外层0, 内层0, 外层1, 内层0, 外层2, 内层0
```
